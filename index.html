Anthony R. King Project Web-Site

Managed and deployed using git/GitHub, from 'gh-pages' branch, in order to be able to work alongside a user site.

Performing Updates

Checkout the 'main' branch, make changes to it, commit the changes, checkout the 'gh-pages' branch, merge the main branch, then push to the 'gh-pages' branch:

  • git checkout main
  • Make changes
  • git add .
  • git commit -m "Updates made"
  • git push origin main
  • git checkout gh-pages
  • git merge main
  • git push origin gh-pages
